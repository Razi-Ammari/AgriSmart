AgriSmart Images Directory
===========================

FAVICON:
--------
Please add your favicon.png file to this directory.
The favicon should be:
- A PNG file named "favicon.png"
- Recommended size: 32x32 or 64x64 pixels
- Transparent background recommended

You can create a favicon using:
- Online tools like favicon.io or favicon-generator.org
- Design software like Photoshop, GIMP, or Canva
- Simple icons representing agriculture/farming

The favicon is already linked in the header.php file and will automatically appear once you add the favicon.png file here.
